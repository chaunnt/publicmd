# Welcome to publicmd

**publicmd** is a lightweight Markdown viewer that renders your `.md` documents as clean HTML in the browser.

## Features

- 📄 Browse all documents from the sidebar
- 🎨 Rendered HTML view with GitHub-style styling
- 🔤 Raw source view for any document
- 💻 Syntax-highlighted code blocks
- 📱 Responsive layout

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Add your `.md` files to the `docs/` folder
4. Start the server with `npm start`
5. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

```bash
# Install dependencies
npm install

# Start the server
npm start
```

The server will start on port **3000** by default. You can change the port with the `PORT` environment variable:

```bash
PORT=8080 npm start
```

## Adding Documents

Drop any `.md` file into the `docs/` directory and it will automatically appear in the sidebar.
